<Warning>
  The **near.org** and **pagoda.co** RPC endpoints are being deprecated. Migration is recommended before **June 1, 2025**. Rate limits will be reduced progressively through August 2025.
</Warning>

NEAR supports multiple third-party RPC providers. Use them for redundancy, load balancing, or archival access. All providers accept standard JSON-RPC 2.0 requests.

```json
{
  "jsonrpc": "2.0",
  "id": "dontcare",
  "method": "<method>",
  "params": { ... }
}
```

<Info>
  All requests require the header `Content-Type: application/json`.
</Info>

## Failover setup

Use `FailoverRpcProvider` from `near-api-js` to automatically fall back to alternate providers if one fails.

<CodeGroup>

```javascript JavaScript
import { FailoverRpcProvider, JsonRpcProvider } from "near-api-js";

const providers = [
  new JsonRpcProvider({ url: "https://free.rpc.fastnear.com" }),
  new JsonRpcProvider({ url: "https://near.drpc.org" }),
  new JsonRpcProvider({ url: "https://near.lava.build:443" }),
];

const failoverProvider = new FailoverRpcProvider(providers);
```

```typescript TypeScript
import { FailoverRpcProvider, JsonRpcProvider } from "near-api-js";

const providers: JsonRpcProvider[] = [
  new JsonRpcProvider({ url: "https://free.rpc.fastnear.com" }),
  new JsonRpcProvider({ url: "https://near.drpc.org" }),
  new JsonRpcProvider({ url: "https://near.lava.build:443" }),
];

const failoverProvider = new FailoverRpcProvider(providers);
```

</CodeGroup>

## Making a raw RPC call

<CodeGroup>

```bash cURL
curl -s -X POST https://free.rpc.fastnear.com \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": "dontcare",
    "method": "status",
    "params": []
  }'
```

```javascript JavaScript
const response = await fetch("https://free.rpc.fastnear.com", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    jsonrpc: "2.0",
    id: "dontcare",
    method: "status",
    params: [],
  }),
});
const data = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://free.rpc.fastnear.com",
    json={
        "jsonrpc": "2.0",
        "id": "dontcare",
        "method": "status",
        "params": [],
    },
)
print(response.json())
```

</CodeGroup>
